Tu vas recevoir une liste de nœuds au format suivant :
{
  "node_id": {
    "title": "nom du nœud",
    "text": "contenu du nœud",
    "available_triggers": {
      "trigger_id": ["contenu du trigger1", "contenu du trigger2", etc..]
    }
  }
}
Pour chaque nœud à traiter, analyse si le "title" ou le "text" contient une référence ou un synonyme au contenu d’un des "available_triggers".
Si c’est le cas pour au moins un champ, considère le trigger comme vrai.

Par exemple pour
sample_nodes = {
    "node_1": {
        "title": "Comment démarrer le moteur",
        "text": "Pour démarrer, appuyez sur le bouton de démarrage.",
        "available_triggers": {
            "trigger_1": ["démarrer le moteur"]
            "trigger_2": ["faire le petit train", "tgv"]
        }
    },
    "node_2": {
        "title": "Un bon repas",
        "text": "Manger un repas de homard.. Miam",
        "available_triggers": {
            "trigger_3": ["cross the ocean"],
            "trigger_4": ["eat a lobster"]
        }
    }
}

Tu devrais renvoyer quelque chose comme:
{
  "node_1": {
    "trigger_1": {
      "trigger_node_id": "trigger_1",
      "should_get_triggered": true,
      "justification": "Le texte du nœud indique comment démarrer un moteur, ce qui est exactement le contenu du trigger."
    },
    "trigger_2": {
      "trigger_node_id": "trigger_2",
      "should_get_triggered": false,
      "justification": "Bien que le bouton de démarrage puisse être interprété comme un « petit train », le contenu du nœud ne mentionne pas explicitement cette activité."
    }
  },
  "node_2": {
    "trigger_3": {
      "trigger_node_id": "trigger_3",
      "should_get_triggered": false,
      "justification": "Le repas de homard ne permet pas de traverser l'océan, même si le mot « ocean » est utilisé dans le contenu du trigger."
    },
    "trigger_4": {
      "trigger_node_id": "trigger_4",
      "should_get_triggered": true,
      "justification": "Le nœud décrit un repas de homard, ce qui est exactement le contenu du trigger."
    }
  }
}

Ta réponse doit respecter strictement le format JSON suivant, compatible avec le modèle Pydantic ci-dessous :
class TriggerEvaluation(BaseModel):
    trigger_node_id: str
    should_get_triggered: bool
    justification: str

class NodeTriggerEvaluations(BaseModel):
    trigger_evaluations: Dict[str, TriggerEvaluation]

class TriggerEvaluationBatch(BaseModel):
    nodes: Dict[str, NodeTriggerEvaluations]

Format de réponse attendu :
{
  "node_id": {
    "trigger_id": {
      "trigger_node_id": "trigger_id",
      "should_get_triggered": true ou false,
      "justification": "ta justification"
    }
    // autres triggers si présents
  }
  // autres nœuds si présents
}

Pour chaque available trigger, la justification doit être rédigée en français. Elle doit brièvement expliquer l'interprétation du node et doit clairement montrer pourquoi ça devrait enclencher le trigger node ou non.
Si un node a N available triggers, alors la réponse doit contenir la réponse pour les N triggers.
