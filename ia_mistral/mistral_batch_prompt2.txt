Tu vas recevoir une liste de nœuds au format suivant :
{
  "node_id": {
    "title": "nom du nœud",
    "text": "contenu du nœud",
    "available_triggers": {
      "trigger_id": "contenu du trigger"
    }
  }
}
Pour chaque nœud à traiter, analyse si le "title" ou le "text" contient une référence ou un synonyme au contenu d’un des "available_triggers".
Si c’est le cas pour au moins un champ, considère le trigger comme vrai.

Ta réponse doit respecter strictement le format JSON suivant, compatible avec le modèle Pydantic ci-dessous :
class TriggerEvaluation(BaseModel):
    trigger_node_id: str
    should_get_triggered: bool
    justification: str

class NodeTriggerEvaluations(BaseModel):
    trigger_evaluations: Dict[str, TriggerEvaluation]

class TriggerEvaluationBatch(BaseModel):
    nodes: Dict[str, NodeTriggerEvaluations]

Format de réponse attendu :
{
  "node_id": {
    "trigger_id": {
      "trigger_node_id": "trigger_id",
      "should_get_triggered": true ou false,
      "justification": "ta justification"
    }
    // autres triggers si présents
  }
  // autres nœuds si présents
}

Pour chaque available trigger, la justification doit être rédigée en français. Elle doit brièvement expliquer l'interprétation du node et doit clairement montrer pourquoi ça devrait enclencher le trigger node ou non.
Si un node a N available triggers, alors la réponse doit contenir la réponse pour les N triggers.
